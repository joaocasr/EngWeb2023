# Engenharia Web 2023

### Fazer setup de um package npm
````
npm init 
````
### Instalar JSON-Server 
````
sudo npm i json-server --save g
````
### Inicializar JSON-Server para simular rest api
````
json-server --watch <'json-file'>
````

### Instalar Axios  
````
npm install axios
````
